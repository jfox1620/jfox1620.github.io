Hi. I'm Justin Fox.

About me: I am an innovative data analyst based in California who loves building strategic solutions that bring value to others. I enjoy the challenge of solving problems, optimizing processes, and leaving things in a better state than when I found it.

I hold a Bachelor's in Mathematics, a Microsoft Power BI Data Analyst Associate certification, and have over eight years of experience building a strong analytical and technical capacity. My ambition is to continue learning and growing to become a professional expert in data science, working with a multitude of various projects, as well as being a leader that inspires others.

My specialties

This is my abridged list of skills and proficiencies.

Data Visualization

I have created many visualizations and dashboards to various stakeholders and levels of staff. I know how to choose the right visuals, how to design them for their audience, and how to emphasize what is important while preventing information overload.

Data Processing

Unlike most data analysts/scientists, I love digging through data and cleansing it. My years of fixing bad data began with Excel tricks and formulas, and evolved to using efficient SQL as well as Python.

BI/Data Engineering

I try to automate the extraction, transformation, and delivery of informative BI solutions wherever possible. The automated reports I have created for recipients minimize the risk of user error, and are built to handle unforeseen data changes.

Machine Learning

Predictive modeling through machine learning is my favorite way to bring value out of data. As a mathematician, I also love the theory and logic behind the algorithms. I have constructed both regression and classfication supervised models using real-world data.

Innovation

Whether or not it is cliché to say I am innovative, the fact remains that I find greatest motivation out of coming up my own outside-the-box ideas and working hard on them to achieve certain goals. In fact, most of the work I have done in my current role as a business analyst were originated from ideas I created.

Statistics

I am intrigued by both descriptive and inferential statistics, as well as probability. Statistical techniques are implemented in my research and analyses to help me make sense of the data at large and present the results effectively.

My most recent projects

Here is a short list of some personal projects I completed recently. All of these are listed on my Github or Tableau Public account.

Exploratory Data Analysis of Open Powerlifting Data in Python

In this project I wrote Python in a Jupyter Notebook that pulled data from the internet on Powerlifting competitions. I preprocessed the data, engineered additional features, and then created some visualizations to help me analyze for insights.

Interactive Power BI Report on Powerlifting Data

This is a Power BI report I developed based off the processed data created from the notebook in the Powerlifting EDA project. It shows a high-level overview of the various data fields with slicers for interactivity.

Cryptocurrency API Data Pipeline

The CoinMarketCap website which provides live information on cryptocurrencies has an API which this program relies on. A python script calls this API, transforms data types and removes some unwanted fields, and loads it into a local database. I added this to the Windows task scheduler to run a few times daily.
