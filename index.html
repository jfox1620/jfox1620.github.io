Hi. I'm Justin Fox.

About me: I am a data professional based in California who turns complex data into actionable insights and scalable systems. I specialize in building robust data pipelines, integrating APIs, automating workflows, and creating interactive dashboards that empower teams and organizations to make smarter, faster decisions.

Currently, I lead the Research & Evaluation (R&E) team at San Joaquin County Probation, where I design and oversee end-to-end data engineering and analytics projects. I’ve built a vast library of automated Python and SQL scripts to ingest, transform, and load multi-source data into a centralized SQL Server data warehouse, eliminating manual processes and enabling faster, more accurate reporting. I also guide the development of dynamic Power BI dashboards and comprehensive annual evaluation reports, turning raw data into clear, actionable insights for leadership and stakeholders.

Beyond my technical expertise, I have experience mentoring and leading teams, redesigning workflows, and streamlining data systems to improve efficiency and data quality. I also volunteer as a Data Strategy & Analytics Lead for Jesus Said Go, optimizing Zoho CRM and creating reports and dashboards that help project managers, donors, and staff make better decisions.

I hold a Bachelor's degree in Mathematics and a Microsoft Power BI Data Analyst Associate certification, with over a decade of experience in analytics, reporting, and automating data processes. I am passionate about advancing as a data engineer because I enjoy solving complex challenges, desire to grow in new technologies, and find satisfaction delivering solutions for others to better leverage their data assets.

Project Showcase

Here is a short list of some personal and work projects I have completed. All of these are listed on my Github.

Nonprofit Clickstream Analytics

A real-world clickstream data pipeline for a nonprofit website that ingests anonymized user events via Google Cloud Pub/Sub, persists raw data in Firestore, and delivers an analytics-ready dataset in BigQuery for visualization and engagement insights using a Looker dashboard.

USGS Earthquakes Lakehouse Pipeline

A full production-grade lakehouse pipeline using PySpark and Delta Lake in Databricks that ingests real-time earthquake data from the USGS API, lands the raw JSON into a structured Bronze -> Silver -> Gold architecture, and produces analytical tables suitable for dashboards, reporting, and machine learning.

Etsy Shop Data Pipeline

ETL pipeline that syncs orders and fulfillments from my Etsy business into Databricks SQL using APIs, normalizing JSON into analytics-ready tables. Demonstrates modular, production-style design and full table replacement.

Exploratory Data Analysis of Open Powerlifting Data in Python

In this project I wrote Python in a Jupyter Notebook that pulled data from the internet on Powerlifting competitions. I preprocessed the data, engineered additional features, and then created some visualizations to help me analyze for insights. I also created a Power BI report.

Predicting Probation Success Using Case Notes

I Developed a machine learning model to predict probation outcomes using natural language processing on case notes, enabling identification of individuals at higher risk of non-compliance.

Pretrial Release Decisions and Failure-to-Appear Analysis

Conducted an A/B-style analysis of pretrial release decisions using chi-squared testing to compare FTA rates, while cleaning and visualizing demographic and risk score data to reveal actionable policy insights.

AB109 Annual Evaluation Report

This is the first comprehensive evaluation I designed and developed which delivers an in-depth statistical overview and analysis of the Public Safety Realignment (AB109) performance metrics in Probation. I went on to oversee a team that developed similar reports for other divisions.
